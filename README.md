# GiftWise

GiftWise is a Ruby on Rails application for managing gift planning, including user profiles, recipients, and events.

## Requirements
- Ruby 3.3.8  
- Rails 7.1.6  
- Bundler  
- SQLite3 (development)

## Setup

Clone the project:

```bash
git clone https://github.com/<your.git](https://github.com/hjmjohnsonSELT2025/giftwiseproject-selt_2025_team_01.git
cd giftwiseproject-selt_2025_team_01

rails db:create
rails db:migrate
rails server
